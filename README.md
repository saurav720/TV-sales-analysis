
# 📊 Electronics Sales Analysis Dashboard

## 📌 Project Overview

This project analyzes the sales performance of three major electronics brands—**Samsung, LG, and Sony**—from **2022 to 2024**. The dashboard provides insights into revenue, profit, returns, sales trends, and brand performance using interactive visualizations.

The objective of this project is to help businesses monitor sales performance, identify profitable brands, analyze return rates, and make data-driven decisions.

---

## 📂 Dataset Information

The dataset contains monthly sales data from **January 2022 to December 2024**.

### Features

| Column         | Description                 |
| -------------- | --------------------------- |
| Year           | Sales Year                  |
| Month          | Sales Month                 |
| Brand          | Product Brand               |
| Cost_per_Unit  | Manufacturing Cost per Unit |
| Units_Sold     | Total Units Sold            |
| Selling Price  | Selling Price per Unit      |
| Revenue        | Total Revenue Generated     |
| Return_Units   | Number of Returned Units    |
| Return_Revenue | Revenue Lost Due to Returns |
| Net_Units_Sold | Units Sold After Returns    |
| Net_Revenue    | Revenue After Returns       |
| Total_Cost     | Total Manufacturing Cost    |
| Net_Profit     | Profit After Returns        |
| Profit         | Profit Per Unit             |

## 🔍 Key Insights

- Sony generated the highest overall revenue.
- Samsung maintained consistent monthly sales performance.
- LG achieved balanced profitability throughout the period.
- Sales increased steadily from January to December each year.
- Return units remained relatively low compared to total units sold.
- Net revenue followed a consistent upward trend across all three years.
